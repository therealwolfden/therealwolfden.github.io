Steam power cycles

Rankine Cycle

Rankine Cycle uses water as a working fluid and uses the components of a pump, boiler, steam turbine and condenser.
  • Operates under a steady flow condition.
  • There is phase change between liquid and vapor.
Process 1 → 2:
  • Isentropic work addition in a pump increases the pressure.
  • Contains compressed liquid.
  • wP = ẆP⁄ṁ = h2 - h1
  • For an ideal pump: wPs = h2s - h1 = v1(p2 - p1)
  • For an actual pump: wP = h2 - h1 = v1(p2 - p1)⁄np
Process 2 → 3:
  • Constant pressure heat addition in a boiler increases the specific volume.
  • Contains compressed liquid to superheated vapor.
  • qin = Q̇in⁄ṁ = h3 - h2
Process 3 → 4:
  • Isentropic expansion in a steam turbine generates the work output.
  • Contains superheated vapor that may turn to a saturated liquid-vapor mixture.
  • wT = ẆT⁄ṁ = h3 - h4
  • For an ideal turbine: wTs = h3 - h4s
    • s3 = s4s
    • p3 = p4s
  • For an actual turbine: wT = h3 - h4 = nT(h3 - h4s)
Process 4 → 1:
  • Constant pressure heat rejection in a condenser lowers the specific volume.
  • Contains a vapor state that returns to a compressed liquid.
  • qout = Q̇out⁄ṁ = h4 - h1
  • To determine what state the working fluid begins as:
    • Start with s4s = s3
    • Look up sf at p4 = sg at p4
    • If sf at p4 < s4s < sg at p4 then the mixture is a saturated liquid-vapor.
      • Determine the quality: X = (s4s - sf at p4) ⁄ sfg at p4
      • Determine the h4s: h4s = hf at p4 + X4shfg at p4
    • If sg at p4 < s4s then use superheated steam tables at p4.
    • h4 = h3 - nT(h3 - h4s)
Thermal Efficiency:
  • nth = 1 - qout⁄qin = 1 - h4 - h1⁄h3 - h2
Properties of Two-Phase Mixtures:
  • Compressed Liquids:
    • Compressed Liquids rely on the saturated liquid-vapor tables (A-4 Saturated Tables).
    • f is saturated liquid, g is saturated vapor.
    • Specific Volume: v(T,p) = vf(T)
    • Specific Internal Energy: u(T,p) = uf(T)
    • Specific Enthalpy:h(T,p) = hf(T) + (p - psat(T)) * vf(T)
    • hf(T) = uf(T) + psat(T)vf(T)
  • Saturated Liquid Vapor Mixtures:
    • f is saturated liquid, g is saturated vapor.
    • Quality: X = mg⁄mtotal
    • Specific Volume: v = (1 - X)vf + Xvg = vf + Xvfg
    • Specific Internal Energy: u = (1 - X)uf + Xug = uf + Xufg
    • Specific Enthalpy:h = (1 - X)hf + Xhg = hf + Xhfg
    • Specific Entropy:s = (1 - X)sf + Xsg = sf + Xsfg
    • hf(T) = uf(T) + psat(T)vf(T)
Back Work Ratio:
  • BWR = wp⁄wt
Reheating:
  • qin = Q̇in⁄ṁ = (h3 - h2) + (h5 - h4)
  • wT = ẆT⁄ṁ = (h3 - h4) + (h5 - h6)
  • qout = Q̇out⁄ṁ = (h6 - h1)
  • nth = wnet, out⁄qin = 1 - qout⁄qin = 1 - (h6 - h1)⁄(h3 - h2) + (h5 - h4)
Regeneration (Open Feedwater Heater):
  • ṁ = ṁ3 = ṁ4 = ṁ5
  • ṁ6 = yṁ
  • ṁ7 = ṁ1 = ṁ2 = (1 - y)ṁ
  • p2 = p6 = p3
  • y = (h3 - h2)⁄(h6 - h2)
    • y is the fraction of steam removed in the open feedwater heater
  • wP = ẆP⁄ṁ = (1 - y)(h2 - h1) + (h4 - h3)
  • qin = Q̇in⁄ṁ = h5 - h4
  • wT = ẆT⁄ṁ = (h5 - h6) + (1 - y)(h6 - h7)
  • qout = Q̇out⁄ṁ = (1 - y)(h7 - h1)
  • nth = 1 - qout⁄qin = 1 - (1 - y)(h7 - h1)⁄h5 - h4
